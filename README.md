# LibraryWebMVC

Описание

Веб-приложение для контроля и учёта книг

Установка

Шаги:

1. Клонируем репозиторий
    ```
   git clone https://github.com/Aferist958/LibraryWebMVC.git
   ```
2. Запуск
    Для запуска должен быть включён Docker Desktop
    ```
   cd .\LibraryWebMVC\
   docker-compose up 
   ```
